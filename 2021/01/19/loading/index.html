网站加载动画

YFun 2021年1月19日 共 79 字 预计 1 分钟

首先，前往主题配置文件中修改：

theme:
    loading: true # 设为 True。

随后，您的网站便有了加载页面时的动画。

动画会遮盖屏幕上的内容，并使用 JavaScript 在页面完全加载后将其隐藏。

因此，此功能不适用于 禁用 JavaScript 的浏览器。
